SOBRE O SISTEMA ESPECIALISTA
  -- Nome:    Sistema Especialista para Escolha de Algoritmo do scikit-learn V.1
  -- Autores: Lincoln Vinicius Schreiber
  -- Resumo:
     O seguinte sistema especialista tem como objetivo identificar qual melhor algoritmo para se usar com a biblioteca scikit-learn para a linguagem de programação Python. Essa biblioteca tem aproximadamente 20 algoritmos prontos para utilizar. 
     
     Cada algoritmo um tem requisitos e funções especificas (regressão, classificação, redução de dimensão) e a ideia é a pessoa responder rapidamente algumas questões sobre o conjunto de dados, e obter uma resposta do melhor algoritmo para utilizar dessa biblioteca. 
     
     Trabalho 1 da disciplina Inteligência Artificial e Sistemas Inteligentes ministrada pela professora doutora Patrícia Augustin Jaques e o professor douto Sandro José Rigo, no semestre 2020/1.

Operador de maior precedência: conjunção
Fator de confiança mínimo para aceitação de regra: 50
